STEP-BY-STEP EXAMPLE

Let's say you want to train a "cyberpunk style" LoRA:

STEP 1: Setup (first time only)
  ‚Üí Run the launcher
  ‚Üí Go to Installation tab

  ‚Üí Go to LoRA Training tab
  ‚Üí Set name: "cyberpunk_style"

  ‚Üí Find 50-100 cyberpunk images

  ‚Üí Open command prompt/terminal
  ‚Üí Run: python train_lora.py --config ~/DiffSynth-Studio/loras/cyberpunk_style/config.json
  ‚Üí Wait 4-24 hours depending on GPU

STEP 6: Use your trained model
  ‚Üí Load base Z-Image-i2 model
  ‚Üí Add your cyberpunk_style LoRA
  ‚Üí Generate cyberpunk-style images!

================================================================================
FREQUENTLY ASKED QUESTIONS
================================================================================

Q: Do I need a GPU?
A: No, but it's 100x faster. CPU training still works.

Q: How much storage do I need?
A: 30GB minimum (20GB for models, 10GB for OS/Python)

Q: Can I train multiple LoRAs?
A: Yes! Create separate configurations for each.

Q: How long does training take?
A: 2-24 hours depending on GPU, dataset size, and settings

Q: What's the best dataset size?
A: 50-200 images for good results

Q: Can I stop and resume training?
A: Yes, checkpoints are saved automatically

Q: How large are trained LoRAs?
A: 50-100MB (very portable!)

Q: Can I use trained LoRAs with other models?
A: Yes, if they use the same base architecture

For more: See INDEX.md - FAQ
